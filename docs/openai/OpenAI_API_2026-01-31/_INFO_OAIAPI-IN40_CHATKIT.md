## Summary

ChatKit provides embeddable UI components for building chat interfaces powered by OpenAI. It simplifies frontend development by providing pre-built, customizable chat widgets.

## Key Facts

- **Status**: Beta [VERIFIED]
- **Purpose**: Embeddable chat UI components [VERIFIED]

## Quick Reference

### Endpoints

- `POST /v1/chatkit/widgets` - Create widget
- `GET /v1/chatkit/widgets` - List widgets
- `GET /v1/chatkit/widgets/{id}` - Get widget
